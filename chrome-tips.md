# chrome快捷键

|操作|快捷键|
|----|----|
|切换页面|ctrl + tab|
| 在隐身模式下打开新窗口|Ctrl+Shift+N |
| 从后台在新标签页中打开链接|按住 Ctrl 键的同时点击链接|
|在新窗口中打开链接|按住 Shift 键的同时点击链接|
|关闭当前窗口|Alt+F4 |
|关闭当前标签页或弹出窗口|Ctrl+W 或 Ctrl+F4 |
|打开“工具”菜单|Alt+F 或 Alt+E |
|从剪贴板中粘贴内容(不带格式)|Ctrl+Shift+V |
|打开新的标签页 | ctrl + t|
|刷新标签页| F5 或ctrl + r|
|调出清除缓存窗口|Ctrl+Shift+Delete|
|关闭标签页|ctrl + w|
